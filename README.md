# How to play:

1. Follow [this link](https://htmlpreview.github.io/?https://raw.githubusercontent.com/peterdeford/bingo/master/sermon_bingo.html) to get to the game board.
2. Hit the `RESET` button to shuffle your squares to a configuration you like.
3. Touch or click squares when Scott mentions/does them.
4. Aim for Bingo!
